SudokGO Overview and Rules

SudokGO is a Sudoku web app with single player and multiplayer(Battle Mode) functionality. The objective of a Sudoku game is to fill a 9x9 grid with digits 1 to 9 so that each column, row, and each of the nine 3x3 sub-grids contain all of the digits from 1 to 9. Therefore, there cannot be any repeat digits in a column, row or sub-grid.

Single Player

Our application first provides a user with the options of playing an easy, medium, or hard single player Sudoku game. Once the user chooses the difficulty, the Sudoku board will be filled and the timer will start. The goal for the user is to finish the Sudoku puzzle as fast as possible. While playing the game, a user can undo/redo a move. A user also can pause the game, which stops the timer and makes the board disappear so that the player is not given an unfair advantage when the timer is stopped. We also allow the user to switch between numbers and marks while filling in the board. The marks are smaller numbers that can be used as notes if the user is unsure of a few numbers that could be placed in one box.
