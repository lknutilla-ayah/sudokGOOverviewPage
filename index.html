SudokGO Overview and Rules

SudokGO is a Sudoku web app with single player and multiplayer(Battle Mode) functionality. The objective of a Sudoku game is to fill a 9x9 grid with digits 1 to 9 so that each column, row, and each of the nine 3x3 sub-grids contain all of the digits from 1 to 9. Therefore, there cannot be any repeat digits in a column, row or sub-grid.

Single Player

Our application first provides a user with the options of playing an easy, medium, or hard single player Sudoku game. Once the user chooses the difficulty, the Sudoku board will be filled and the timer will start. The goal for the user is to finish the Sudoku puzzle as fast as possible. A user also can pause the game, which stops the timer and makes the board disappear so that the player is not given an unfair advantage when the timer is stopped.

Board Entry

During play, the user can switch between two entry modes, "number" and "mark". "Number" entries get checked against the board for conflicts, and only one can be entered per box. On the other hand, "marks are not validated and multiple "marks" can be entered for a single box and are used as notes if the user is unsure about an entry and wants to remember potential numbers. Finally, a user can undo/redo numbers or marks from a box.

Board Navegation

To assist play, SudokGo allows a user multiple ways navigate the board. In general, the user can use the mouse to click any box, editable or non-editable, and the box is highlighted. Once the user has clicked into the board, he or she also can use the arrow keys to quickly move through the board. SudokGO also works cross platform and entries can be logged by keyboard or using the numpad to enter and delete numbers and marks

Multiplayer (Battle Mode)

SudokGO also includes a battle mode, which is a multiplayer version of Sudoku. If the user plays in battle mode, they will be given a link to send to his or her friend. Once the friend has joined the game, our application asks them if they are ready. Once both players are ready, the multiplayer game starts and the goal is to finish the same Sudoku puzzle faster than the opponent. If a player puts the correct number in a box, then the corresponding box on the other player's board is highlighted in yellow. As competitors see the board fill up with yellow, the intensity increases and the race is on to see who can finish first.

Leaderboard

Once a single player user completes a game, the user can enter their name, which will be put on a leaderboard if their time is one of the top ten times.
